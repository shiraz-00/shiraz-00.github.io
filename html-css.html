The Difference Between Margins, Borders and Padding in CSS



Margins:
Margins control the spacing between the element and the edge of you webpage. Margins can be used to change the position of the element, or to add space between multiple elements.

Borders:
Borders are essentially the line that seperates the margin and padding.
